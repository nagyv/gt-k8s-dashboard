
## Installation

```st
Metacello new
	repository: 'github://nagyv/gt-k8s-dashboard:main/src';
	baseline: 'GtK8sDashboard';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfGtK8sDashboard asClass loadLepiter
```
